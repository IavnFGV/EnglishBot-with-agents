## Fast path
- New command or command text: `command_registry.py`, `i18n.py`, matching handler, matching tests.
- New learner flow bug: start with `training.py` or `homework.py`, then the matching handler/dialog tests.
- Authoring bug: start with `teacher_content.py` plus the dialog/handler test pair.
- Persistence bug: `db.py` plus the narrow domain module that owns the affected table.
